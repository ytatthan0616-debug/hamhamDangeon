# ハムハムダンジョン (hamhamDungeon)

## 概要
『ハムハムダンジョン』は，主人公のハムスターが迫りくる敵の群れをなぎ倒していく，WAVE制のダンジョンサバイバル＆育成ゲームです．
クリックや距離に応じた自動追尾・攻撃システムを搭載しており，大群が入り乱れる「わちゃわちゃ感」のある乱戦バトルを楽しむことができます．

## 🎮 ゲームの特徴
* **WAVE制サバイバル戦闘**
  次々と湧いてくる敵を倒し，WAVEを進めていきます．WAVEが進むごとに敵のステータスが上昇し，ボスキャラクターも登場します．
* **最適化された「わちゃわちゃ」乱戦システム**
  キャラクター同士の物理的な押し出し（物理演算の競合）を排除し，敵とプレイヤーが入り乱れるスムーズな乱戦挙動を実現しています．
* **超軽量化システム**
  毎フレームの重い検索処理（`FindGameObjectsWithTag`など）を廃止し，独自の「名簿管理システム」を実装．大量の敵やエフェクトが出現してもサクサク動作します．
* **ド派手なエフェクトと高品質な2D描画**
  URP（Universal Render Pipeline）を採用．Cartoon FXを活用した派手なスキルエフェクトや，専用シェーダーによる美麗な描画が特徴です．

## 💻 開発環境
* **Game Engine**: Unity 6 (6000.4.8f1)
* **Render Pipeline**: 2D URP (Universal Render Pipeline)
* **Platform**: Windows / Mac / Linux

## 📂 ディレクトリ構成
* `Assets/`: スクリプト，プレハブ，画像，マテリアル等の主要データ
* `Packages/`: Unityパッケージデータ
* `ProjectSettings/`: URP設定，Input設定等のプロジェクト全体設定

## ⚠️ 開発者向けノート（クローン・同期時の注意点）
別PCでクローン（Pull）した際，グラフィック設定が初期化され，エフェクトが透明になったりUIの背景がおかしくなる場合があります．
その場合は，以下の手順でURPを再設定してください．

1. `Edit` > `Project Settings` > `Graphics` を開く．
2. `Scriptable Render Pipeline Settings` にプロジェクト内の `URPAsset` を再割り当てする．
